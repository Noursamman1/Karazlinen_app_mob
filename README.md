# Karaz Linen Mobile Commerce

Production foundation for the Karaz Linen mobile commerce platform.

Architecture baseline:

- Flutter mobile app for iOS and Android
- Mobile BFF/API Gateway
- Magento 2 as commerce source of truth

Execution notes:

- This repository is bootstrapped for parallel scoped implementation.
- Canonical API contracts live under `contracts/`.
- Architecture decisions live under `adr/`.
- Per-agent deliverables live under `artifacts/`.
